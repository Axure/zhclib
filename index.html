ZhCLib

A C library intended for generic and handy usage.

Download this project as a .zip file Download this project as a tar.gz file

Get Started

This is a C library providing useful method calls that standard C does not provide in handy.

This library is implemented in an Object Oriented way, that basic types and string (char *) are primitives, and others are subclasses of Objects, while some grouped utility method calls are named after the rule of member methods.

For users who only want to do things in the pure C way, you may still find the string part handy in your development.
